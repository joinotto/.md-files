FAQs
----

### When is a model release form needed?



A model release form is required when using an individual's image for commercial purposes, such as advertising, stock photography, or promotional content, to avoid legal issues.

### Who signs a model release form?



The model (or their legal guardian if under 18) signs the form, granting the photographer or company rights to use their image for agreed-upon purposes.

### Can a model revoke a signed release?



Generally, a model release is legally binding once signed, but specific terms may allow revocation if both parties agree or if misrepresentation occurred.

### Do I need a model release for editorial use?



No, a model release is usually not needed for editorial use in news or documentaries, but it is required for commercial use like advertising or marketing.

### What happens if I use someone's image without a model release?



Using an image without consent for commercial purposes can lead to legal action, including lawsuits for privacy violations or unauthorized use.
