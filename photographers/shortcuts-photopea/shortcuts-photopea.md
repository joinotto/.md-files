Why Use Shortcuts?
------------------

Utilizing keyboard shortcuts not only speeds up the photo editing process but also enhances your overall productivity. Artists and editors often face time constraints, and shortcuts can save valuable seconds that add up to minutes or hours in larger projects.

Common Shortcuts in Photopea
----------------------------

### Navigation Shortcuts

*   Zoom In/Out: 
    *   Ctrl + "+" to zoom in
    *   Ctrl + "-" to zoom out
    *   Ctrl + 0 to fit screen

*   Pan: 
    *   Hold Spacebar and drag

*   View Modes: 
    *   Ctrl + 1 for 100%
    *   Ctrl + 2 for Fit to screen
    *   Ctrl + 3 for actual pixels

### Selection Shortcuts

*   Marquee Tool: 
    *   M to select the Rectangular Marquee Tool

*   Lasso Tool: 
    *   L to select the Lasso Tool
    *   Shift + L to cycle through Lasso Tools

*   Select All: 
    *   Ctrl + A

*   Deselect: 
    *   Ctrl + D

### Layer Shortcuts

*   New Layer: 
    *   Ctrl + Shift + N

*   Duplicate Layer: 
    *   Ctrl + J

*   Hide/Reveal Layer: 
    *   Click on the eye icon in the layers panel

*   Merge Layers: 
    *   Ctrl + E

### Editing Shortcuts

*   Undo/Redo: 
    *   Ctrl + Z to Undo
    *   Ctrl + Shift + Z to Redo

*   Copy/Paste: 
    *   Ctrl + C to Copy
    *   Ctrl + V to Paste

*   Transform: 
    *   Ctrl + T to enter free transform mode

Advanced Shortcuts for Efficient Workflow
-----------------------------------------

Once you are comfortable with basic shortcuts, consider incorporating these advanced shortcuts into your routine.

### Tool Shortcuts

*   Brush Tool: 
    *   B for Brush Tool
    *   Shift + B to cycle through Brush Tools

*   Eraser Tool: 
    *   E for Eraser Tool
    *   Shift + E to cycle through Eraser Tools

*   Text Tool: 
    *   T for Text Tool

### Color Shortcuts

*   Reset Colors: 
    *   D to reset foreground and background colors

*   Switch Colors: 
    *   X to switch foreground and background colors

### File Shortcuts

*   Open File: 
    *   Ctrl + O

*   Save File: 
    *   Ctrl + S

*   Export File: 
    *   Ctrl + Shift + S

Customizing Your Shortcuts
--------------------------

Photopea allows customization of some shortcuts to better suit your workflow:

*   Navigate to Edit > Keyboard Shortcuts.
*   Modify existing shortcuts or set new ones for your frequently used tools.
