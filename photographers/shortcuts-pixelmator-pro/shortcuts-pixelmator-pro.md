Essential Shortcuts for Beginners
---------------------------------

### 1. Basic Navigation Shortcuts

*   Open Project: `Cmd + O`
*   New Project: `Cmd + N`
*   Close Project: `Cmd + W`

### 2. Canvas Manipulation Shortcuts

*   Zoom In: `Cmd + +`
*   Zoom Out: `Cmd + -`
*   Actual Size: `Cmd + 0`

### 3. Layer Management Shortcuts

*   New Layer: `Cmd + Shift + N`
*   Duplicate Layer: `Cmd + J`
*   Delete Layer: `Backspace`

These shortcuts provide a foundation for beginners to start navigating Pixelmator Pro confidently.

Intermediate Shortcuts for Enhanced Efficiency
