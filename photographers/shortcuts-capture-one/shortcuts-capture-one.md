What are Capture One Shortcuts?
-------------------------------

Capture One shortcuts are keyboard combinations that allow you to execute commands quickly without navigating through menus. Learning these shortcuts can drastically reduce editing time, enhance creativity, and streamline your workflow.

Why Use Keyboard Shortcuts?
---------------------------

### Improved Efficiency

Using shortcuts allows you to apply adjustments and edits faster, saving precious time during your workflow.

### Enhanced Focus

Relying on your keyboard helps minimize distractions from the mouse, allowing you to concentrate more on your creative process.

### Increased Consistency

Shortcuts lead to fewer errors, ensuring that the same edits can be quickly replicated across various images.

Essential Capture One Shortcuts
-------------------------------

### Navigation

*   Arrow Keys: Move between images.
*   Cmd + 1-4 (Mac) / Ctrl + 1-4 (Windows): Switch between different workspaces.

### Basic Functions

*   Cmd + S (Mac) / Ctrl + S (Windows): Save your session.
*   Cmd + Z (Mac) / Ctrl + Z (Windows): Undo your last action.

### Editing Tools

*   B: Brush tool for local adjustments.
*   E: Erase brush, useful for refining adjustments.
*   M: Crop tool, allowing for quick image framing.

### Adjustments

*   Cmd + L (Mac) / Ctrl + L (Windows): Adjust levels.
*   Cmd + C (Mac) / Ctrl + C (Windows): Copy adjustments.
*   Cmd + V (Mac) / Ctrl + V (Windows): Paste adjustments to another image.

### Exporting

*   Cmd + R (Mac) / Ctrl + R (Windows): Start the export dialogue.
*   Cmd + Shift + D (Mac) / Ctrl + Shift + D (Windows): Export images with a given preset.

Intermediate Shortcuts
----------------------

Once you're comfortable with the basics, try to incorporate these intermediate shortcuts:

### Layer Management

*   Cmd + T (Mac) / Ctrl + T (Windows): Toggle the Layer tool.
*   Cmd + N (Mac) / Ctrl + N (Windows): Create a new layer for adjustments.

### Color Grading

*   Cmd + Shift + C (Mac) / Ctrl + Shift + C (Windows): Open Color Grading tab for advanced editing.

Creating Custom Shortcuts
-------------------------

Capture One allows customization of shortcuts, enabling you to tailor the editing process to your preferences. Here’s how:

1.   Go to the menu and select Edit > Keyboard Shortcuts.
2.   Choose commands you frequently use and assign new shortcuts.
3.   Save your custom setup for future use.

Best Practices for Learning Shortcuts
-------------------------------------

### Practice Regularly

Commit to using shortcuts daily to reinforce memory retention.

### Use Cheat Sheets

Print out a list of your most used shortcuts and keep it near your workspace.

### Keep It Simple

Start with a few shortcuts, then gradually increase as you become more comfortable.
