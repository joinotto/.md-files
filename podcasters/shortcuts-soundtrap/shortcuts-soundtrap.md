### Keyboard Shortcuts

Much of the efficiency of using Soundtrap comes from utilizing keyboard shortcuts. Here’s a breakdown of the most vital ones:

#### 1. General Shortcuts

*   Play/Pause: Spacebar
*   Stop: Enter
*   New Project: Ctrl + N (Cmd + N on Mac)
*   Save Project: Ctrl + S (Cmd + S on Mac)

#### 2. Navigation Shortcuts

*   Zoom In: Ctrl + Up Arrow (Cmd + Up Arrow on Mac)
*   Zoom Out: Ctrl + Down Arrow (Cmd + Down Arrow on Mac)
*   Scroll Left: Alt + Left Arrow
*   Scroll Right: Alt + Right Arrow

#### 3. Editing Shortcuts

*   Cut: Ctrl + X (Cmd + X on Mac)
*   Copy: Ctrl + C (Cmd + C on Mac)
*   Paste: Ctrl + V (Cmd + V on Mac)
*   Undo: Ctrl + Z (Cmd + Z on Mac)
*   Redo: Ctrl + Shift + Z (Cmd + Shift + Z on Mac)
