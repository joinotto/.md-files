Key Zoom Shortcuts You Should Know
----------------------------------

Understanding how to use Zoom’s shortcuts will not only streamline your online meetings but also showcase your proficiency with the platform. Here’s a comprehensive breakdown by function:

### 1. Meeting Controls

#### Zoom Meeting Shortcuts (Windows)

*   Start/Stop Video: Alt + V
*   Mute/Unmute Audio: Alt + A
*   Share Screen: Alt + S
*   End Meeting: Alt + Q

#### Zoom Meeting Shortcuts (Mac)

*   Start/Stop Video: Command + Shift + V
*   Mute/Unmute Audio: Command + Shift + A
*   Share Screen: Command + Shift + S
*   End Meeting: Command + W

### 2. Navigation Shortcuts

Efficient navigation is crucial, especially during busy meetings or presentations. Use these to streamline your experience:

*   Switch to Gallery View: Alt + Shift + F (Windows) / Command + Shift + F (Mac)
*   Switch Between Screens: Alt + Tab (Windows) / Command + Tab (Mac)
*   Screen Sharing Pause/Resume: Alt + T (Windows) / Command + T (Mac)

### 3. Chat Functions

Communicating in chat can complement verbal discussions. Here are shortcuts to simplify texting in Zoom:

*   Open Chat: Alt + H (Windows) / Command + Shift + H (Mac)
*   Send Chat Message: Enter
*   Navigate Chat: Arrow Keys to move up and down.

### 4. Other Useful Shortcuts

#### Accessibility Shortcuts

*   Enable/Disable Closed Captioning: Alt + C
*   Gallery View: Alt + F2 (Windows) / Command + Shift + F2 (Mac)

#### Record Meetings

*   Start/Stop Recording: Alt + R (Windows) / Command + R (Mac)
