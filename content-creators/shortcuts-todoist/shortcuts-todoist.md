Essential Todoist Shortcuts
---------------------------

Here’s a list of some of the most useful Todoist keyboard shortcuts to learn:

### 1. Navigation Shortcuts

*   Ctrl + Q (Windows) / Cmd + Q (Mac): Quickly add a new task.
*   Ctrl + Shift + N: Open a new project.
*   Ctrl + 1-9: Switch between different projects.

### 2. Task Management Shortcuts

*   Enter: Add a task.
*   Ctrl + Enter: Complete a task.
*   Ctrl + Backspace: Delete a task.

### 3. Task Editing Shortcuts

*   Ctrl + R: Add a recurring due date to your task.
*   # : Use a hashtag to categorize or add a label when creating a task.
*   @ : Assign a task to a project.

### 4. Viewing and Organizing Tasks

*   Ctrl + C: Copy selected tasks.
*   Ctrl + V: Paste copied tasks.
*   Ctrl + , (comma): Open Todoist settings.

Tips for Remembering Shortcuts
------------------------------

Learning and remembering shortcuts can sometimes be daunting. Here are some tips:

*   Practice regularly: Use the shortcuts in your daily task management to help them stick.
*   Create a cheat sheet: Write down the most important shortcuts and place them near your workspace.
*   Focus on the essentials first: Start with shortcuts that you use most often, then gradually learn the less frequently used ones.

Common Mistakes to Avoid with Todoist Shortcuts
-----------------------------------------------

It’s important to be aware of common pitfalls users fall into when utilizing shortcuts:

*   Overcomplicating Tasks: Trying to assign too many labels or projects to a task can make your workflow cumbersome.
*   Ignoring Recurring Due Dates: Forgetting to set these can lead to mismanagement of repetitive tasks.
