What is DaVinci Resolve?

Why Use Keyboard Shortcuts?
---------------------------

Keyboard shortcuts reduce reliance on the mouse, allowing for faster and more efficient editing. By mastering these shortcuts, you can dramatically speed up your workflow and focus on your creative process rather than the technicalities of the software.

Essential DaVinci Resolve Shortcuts
-----------------------------------

### 1. Editing Shortcuts

*   Cut: `Control + B` (Windows) / `Command + B` (Mac)
*   Copy: `Control + C` / `Command + C`
*   Paste: `Control + V` / `Command + V`
*   Undo: `Control + Z` / `Command + Z`
*   Redo: `Control + Shift + Z` / `Command + Shift + Z`

### 2. Navigation Shortcuts

*   Go to Next Edit Point: `Down Arrow`
*   Go to Previous Edit Point: `Up Arrow`
*   Zoom In/Out on Timeline: `Control + =`/`Control + -` (Windows) / `Command + =`/`Command + -` (Mac)

### 3. Playback Shortcuts

*   Play/Pause: `Spacebar`
*   Stop: `K`
*   Play Forward Frame by Frame: `L`
*   Play Backward Frame by Frame: `J`

### 4. Color Grading Shortcuts

*   Open Color Panel: `Shift + 6`
*   Toggle between Corrector and Gallery: `Shift + Y`
*   Reset Color Grade: `Control + R` / `Command + R`

### 5. Export Shortcuts

*   Deliver Page: `Shift + 8`
*   Add to Render Queue: `Control + Shift + R` / `Command + Shift + R`
*   Start Render: `Control + Enter` / `Command + Enter`

Workflow Tips to Enhance Efficiency
-----------------------------------

1.    Create Your Own Shortcuts: DaVinci Resolve allows users to customize keyboard shortcuts. Refer to the Keyboard Customization menu (found under the DaVinci Resolve menu) to tailor shortcuts to your needs. 
2.    Practice Regularly: The more you use these shortcuts, the more they will become second nature. Include them in your daily editing tasks to enhance muscle memory. 
3.    Utilize Cheat Sheets: Consider creating a printed cheat sheet or a digital note file containing your most-used shortcuts for quick reference. 
4.    Organize Your Workspace: A well-organized workspace can improve efficiency and reduce distractions. Take the time to set up your layout to suit your editing style.
