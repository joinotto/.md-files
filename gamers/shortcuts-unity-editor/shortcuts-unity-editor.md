Essential Unity Editor Shortcuts
--------------------------------

### Navigating the Scene View

*   W, A, S, D: Pan the Scene view.
*   Q: Switch to the View Tool for navigation.
*   E: Activate the rotate tool.
*   R: Scale objects.
*   F: Frame selected objects.

### Working with GameObjects

*   Ctrl + D (Cmd + D on macOS): Duplicate selected GameObject.
*   Delete: Remove selected GameObject.
*   Ctrl + Z (Cmd + Z on macOS): Undo last action.
*   Ctrl + Y (Cmd + Y on macOS): Redo last action.

### Scene Management

*   Ctrl + S (Cmd + S on macOS): Save current scene.
*   Ctrl + N (Cmd + N on macOS): Create a new scene.
*   Ctrl + Shift + S (Cmd + Shift + S on macOS): Save as.

### Asset Management

*   Ctrl + F (Cmd + F on macOS): Find assets in the project.
*   Right-click + Reveal in Explorer (Finder on macOS): Quickly locate assets on your file system.

### Play and Debug Mode

*   Ctrl + P (Cmd + P on macOS): Play or pause the game.
*   Ctrl + Shift + P (Cmd + Shift + P on macOS): Step through your code while in debug mode.

Customizing Your Shortcuts
--------------------------

Unity also allows you to customize keyboard shortcuts to fit your workflow better. To do this, navigate to Edit > Shortcuts in the menu bar. Here, you can view existing shortcuts and modify them according to your preferences.
